To run this project locally, follow these steps:

1.  **Clone the repository**
    ```bash
    git clone https://github.com/yoanbernabeu/DevChallenges.git
    cd DevChallenges
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Start the development server**
    ```bash
    npm run dev
    ```

4.  **Open your browser**
    Navigate to `http://localhost:4321` to see the project running.
